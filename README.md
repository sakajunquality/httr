httr (HTTp Response)
=================================

Overview
------------
Command line tool for displaying HTTP response headers.




Installation
------------
To install, simply run:

```
$ go get github.com/sakajunquality/httr
```

To install with brew,

```
$ will be avaiable soon (Hopefully)
```



Usage
------------

```
$ httr example.com
$ httr https://example.com
```
